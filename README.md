# CENTRAL-DE-REGULARIDADE

Este repositório contém os arquivos necessários para rodar a aplicação localmente.

## Configuração Local

**Pré-requisitos:** Node.js

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Configure a variável `GEMINI_API_KEY` no arquivo `.env.local` para autenticar com os serviços do backend.
3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
